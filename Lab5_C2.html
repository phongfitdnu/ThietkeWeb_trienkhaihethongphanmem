Giới thiệu về HTML

HTML là viết tắt của HyperText Markup Language.

HTML rất quan trọng trong việc xây dựng trang web.

HTML có thể kết hợp với CSS để tạo ra giao diện đẹp hơn.

Chú thích: Đây là một đoạn văn bản nhỏ.

Một số công thức khoa học: H2O, CO2

Một số công thức toán học: x2 + y2 = z2

Giá cũ: 1.000.000đ Giá mới: 900.000đ

Học HTML rất thú vị!
